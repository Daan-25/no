- `index.html`
- `styles.css`
- `script.js`
- `CNAME`

## Deploy on GitHub Pages
1. Push to `main`.
2. Go to repository `Settings -> Pages`.
3. Choose `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save and wait for deployment.
